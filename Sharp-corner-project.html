How to Create an account on github and how to link it to VS Code

Aryan raj

What is Github ?

Github is a Web-based platform that uses Git for version control and collaboration, primarily for software development.

It allows developers to store, manage, and track changes to their code, and collaborate with others on projects, including the ability to review code.

Essentially, it is social coding platform for developers.

1. How to create an account on Github ?

Here are some steps to create an account on Github :

Step 1 : Go to Github

Visit Github Click here to go to Github

Step 2 : Sign Up

  • Click "Sign up" (top right corner).
  • Enter :
    • Username
    • Email address
    • Password
  • Click "Create Account"

Step 3 : Verify Email

  • Github Will Send a Verification code to Your Email.
  • Enter the code to continue

Step 4 : Set Preferences

  • Choose your Experience level (optional)
  • Pick your interest (optional)
  • Choose Free plan (or Pro, if needed).

Step 5 : Complete CAPTCHA

  • Verify you're not a bot, and proceed.

2. How to Create repository on Github ?

Here are some steps to create a repository on Github :

Step 1 : Create a New Repository

  • Click the + icon on the top-right corner.
  • Select "New Repository"

Step 2 : Fill Repository Details

  • Repository name (e.g.. my-first-repo)
  • Description (optional)
  • Choose
    • Public (anyone can see it)
    • Private (only you can see it)
  • Optionally
    • Check "initialize this repository with a Readme" if you want to include a Readme file.
    • Add a .gitignore (optional)
    • Choose a license (optional)

Step 4 : Create

  • Click "Create Repository"

3. How to link Github With VS Code

Here are some steps to link Github with VS Code :

Step 1 : Open VS Code

Step 2 : Open the Command Pallete

  • Press Ctrl + Shift + P (windows/linux) or cmd +Shift + P (mac)

Step 3 :Sign in

  • Type : Github: sign in
  • Press Enter
  • A browser will open asking you to authorize VS Code to access your Github account.
  • Authorize it and return to VS Code

4. How to Clone a Github Repository with VS Code

Step 1 : Open the terminal in VS Code

  • ctrl +-- (backtick) or from the menu: Terminal>New Terminal

Step 2 : Run this command

  • Command "git clone https://github.com/your-username/your-repo-name.git"

Step 3 : Change into the project directory

  • cd your-repo-name

Step 4 : Open it in VS Code

You're Done

You are Successfully :

  • Linked Github with VS Code
  • Cloned a repository
  • Open the project to start coding
